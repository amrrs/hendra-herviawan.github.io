Content based Recommendation is system that suggesting you item or procudt that might be of interest to you based on anather product content that you already interested (history). Most often, these systems recommend related item based on similarity content such as title, category, product description,etc.

Content based recommendation use actual attributes of the item itself to recommend similar products. This makes a ton of sense, as it's how we actually shop in the real world. We go into the supermarket and look at all the items, which are probably physically arranged on the shelf according to brand, price, or category.

3G External Antena Modem Bolt Slim & Max 3dbi ts9 - e5776

Let say that you interested in above product, from title we know that its "external antena" for 3G Modem. very easy for human to find similar item becouse we can process unstructured data like text, image or sound easely. Computer is very different from human when processing unstructured informatio.

Training a Model¶

import numpy as np
import pandas as pd

from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import linear_kernel

data_source = 'input/Jkt-Index-tiny.csv.zip'
model = 'model/cosine_similarities-20180310'

def train(ds):
    tf = TfidfVectorizer(analyzer='word',
                         ngram_range=(1, 2),
                         min_df=0,
                         stop_words='english')
    tfidf_matrix = tf.fit_transform(ds['title'])
    return linear_kernel(tfidf_matrix, tfidf_matrix)


ds = pd.read_csv(data_source)
np.save(model, train(ds))
